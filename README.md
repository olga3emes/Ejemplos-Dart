# Introducción a Dart: Ejemplos 
### 2ºDAM 2025-2026

## Cómo usar en VS Code
1) Instala el **Dart SDK** y añade la carpeta `bin` del SDK al `PATH`.
2) Abre esta carpeta en VS Code (`Archivo > Abrir carpeta...`).
3) Instala la extensión **Dart** si te la propone.
4) En el terminal integrado, ejecuta cualquiera de los ficheros con:
   ```bash
   dart run bin/<nombre_del_ejemplo>.dart
   ```

> Consejo: Usa "Run" en VS Code cuando te aparezca el botón en cada `main()`.
